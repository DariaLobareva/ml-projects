**Цель проекта:**
Построить несколько моделей машинного обучения для предсказания риска сердечно-сосудистых заболеваний и сравнить их метрики качества моделей

**Описание:**
Учебный ML-проект по предсказанию риска сердечно-сосудистых заболеваний. Реализован полный pipeline: EDA, подготовка данных, обучение и сравнение моделей классификации. Основные ограничения датасета — выраженный дисбаланс классов и ограниченный объём выборки. Для оценки моделей использовались метрики Recall, Precision, F1-score.

**Задачи, которые были решены в проекте:**
-проведен разведочный анализ данных (EDA)
-выполнена подготовка данных перед обучением
-обоснован выбор метрик качества для сравнения моделей
-построены 3 модели классификации
